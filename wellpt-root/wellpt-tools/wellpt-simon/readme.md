## 集成simple monitor，用于跟踪Java性能

[项目地址](https://github.com/virgo47/javasimon)

## 使用篇
### 1、webapp中引入POM
```
        <dependency>
            <groupId>com.wellsoft</groupId>
            <artifactId>wellpt-simon</artifactId>
            <version>1.0</version>
        </dependency>
```

### 2、访问http://${host}:${port}/simon-console/index.html
