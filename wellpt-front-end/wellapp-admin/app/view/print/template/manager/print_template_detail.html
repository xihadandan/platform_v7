  • 模板列表

word模板:一般不建议使用 html模板: 功能最强大（支持freemarker的全语法），但在套打出来后，不可编辑。
WordXml模板:功能也还可以，但没html强大，但套打出来后是可编辑的,可以再次调整内容及格式(不建议使用，建议使用下面这种方式)
WordXml模板(批注): 功能与WordXml模板是一样，不同之处在于:freemarker脚本是写在批注中。(政府收发文建议使用)
Poi-tl模板: 模版文件和嵌套的正文必须为docx格式文件
帮助
Word模板上传2003的版本、应用服务器安装office 2000及jacob-1.17-x86.dll(32位操作系统)或jacob-1.17-x64.dll(64位操作系统)放到系统PATH目录(C:\Windows\System32)下
